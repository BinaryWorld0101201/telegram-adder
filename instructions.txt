Для запуска парсера нужно:
1) Добавить номера телефонов (России) и никнеймы телеграмма в файл с расширением .txt
2) Каждая новая строчка это либо один номер, либо один никнейм
P.S: по-другому программа не будет работать
3) Поместить этот файл нужно в папку database

Для лучшего заруска добавления пользователей в канал:
1) Заранее открыть телеграмм в фоне
2) Хороший интернет
3) После запуска не трогать компьютер


Программа работает только на Windows 10.

Для работы самой программы требуется путь к telegram.exe в папке, где он установлен
1) открыть файл  settings.ini через блокнот
2) в строке " telegram = " указать сам путь без ковычек

Базовое время паузы указано в переменной "sleep" в секундах
    default: sleep = 1800 секунд (30 минутам)


Для запуска программы нужно открыть файл main.bat

Каждый раз программа заного парсит данные из ./database поэтому лучше менять их.
